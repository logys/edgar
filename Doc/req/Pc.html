1.0 Communicación

Se utilizara el protocolo bluetooth disponible en la mayoria de los dispositivos modernos o un dongle usb si el equipo carece del hardware.

2.0 Python

Se elige Python como lenguaje por motivos de facilidad y multiplataforma.

3.0 Windows

Aunque el proyecto se desarrolla en linux fedora, el objetivo es distribuirlo en Windows 10, pues es el Sistema Operativo utilizado en la mayoría de los dispositivos en los laboratorios.
